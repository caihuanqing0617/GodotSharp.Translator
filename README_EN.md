# Godot CSharp RunTime Document Translated to Other Languages

Godot C# Runtime help documentation translated by large language models into various national languages.

## Introduction

-   Based on the translation of the Godot C# Runtime help documentation, you can view the usage and calls of functions in your local language in real-time.
    
## Instructions

-   Find the corresponding language and version.
    
-   For example:
    
  1.  Locate the Chinese folder under the current Git directory.
    
  2.  Find the corresponding version, for instance, v4.6.1.
    
  3.  Find the GodotSharp.?.xml file.
    
  4.  Copy it to:

    - On Windows: C:\Users\<YourUsername>\.nuget\packages\godotsharp\4.6.1\lib\net8.0

    - On macOS: /Users/<YourUsername>/.nuget/packages/godotsharp/4.6.1/lib/net8.0

    - On Linux: /Users/<YourUsername>/.nuget/packages/godotsharp/4.6.1/lib/net8.0
    
  5.  Delete the original GodotSharp.xml file.
    
  6.  Rename the found GodotSharp.?.xml file to GodotSharp.xml and copy it to the above directory.
    
  7.  Done. Open your project, and you should see the language you selected.
