# Godot CSharp RunTime Help Document To Other Language

Godot C# 运行库的帮助文档的大语言模型的翻译到其它国家的语言

## 介绍

-   基于Godot C# 运行库的帮助文档的翻译,可以实时的查看函数本地语言的用法及调用.
    
## 使用说明

-   找到对应的语言和版本
    
-   譬如说:
    
  1.  在当前的Git目录 Chinese 目录下面找
    
  2.  找到对应的版本 如果是 v4.6.1
    
  3.  找到 GodotSharp.?.xml
    
  4.  复制到 C:\\Users\\用户名\\.nuget\\packages\\godotsharp\\4.6.1\\lib\\net8.0
    
  5.  删除 GodotSharp.xml
    
  6.  把找到的 GodotSharp.?.xml 重命名为 GodotSharp.xml
    
  7.  完成.然后打开你的项目就可看到你所对应的语言了
